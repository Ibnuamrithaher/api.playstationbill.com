# PlayStation Rental Cashier Platform API

Backend API for the PlayStation Rental Cashier Platform, built with Go (Golang), Gin, and GORM.

## Requirements
- Go 1.22.0 or later
- PostgreSQL database

## Getting Started
1. Clone the repository.
2. Create a `.env` file based on `.env.example` and fill in the database configuration.
3. Run `go mod tidy` to download dependencies.
4. Run the application:
   ```bash
   go run cmd/server/main.go
   ```
